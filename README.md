# 🏏 Cricket Analytics Web App

This project is a **Streamlit-based Cricket Analytics Application** that allows users to analyze and compare player performance based on match conditions.

---

## 🚀 Features

- Player vs Player comparison
- Phase-based analysis (Powerplay, Middle, Death)
- Bowler-specific performance
- Key metrics:
  - Runs
  - Balls
  - Strike Rate
  - Average
- Graphical visualization
- Automated recommendation of best player
- Top 5 player ranking system

---

## 🧠 Analysis Performed

The system performs:

- Data filtering based on:
  - Batsman
  - Bowler
  - Match phase
- Aggregation:
  - Total runs
  - Balls faced
  - Dismissals
- Metrics calculation:
  - Strike Rate = (Runs / Balls) × 100
  - Average = Runs / Dismissals
- Ranking logic for best players
